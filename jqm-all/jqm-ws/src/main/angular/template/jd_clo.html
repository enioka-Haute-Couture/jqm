Class Loader Options

Specific Isolation Context
If set all job definitions using the same value (case sensitive) will share the same CL. If this parameter is set you need to ensure all job definitions with the same specificIsolationContext share the same configuration for Child First Class Loader and Hidden Java Classes.
Hidden Java Classes
Offers the possibility to hide Java classes from jobs using one or more regex to define classes to hide from the parent class loader
Child First Class Loader
Offers an option to have a child first class loader for this job definition
