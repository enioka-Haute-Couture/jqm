Compatibility matrix
#############################

Java (JRE)
************

Supported: 12, 18, 21.

Databases
**************

HSQLDB: version embedded with JQM only.

Oracle: 10gR2, 11gR2, 12cR2, 19c.

DB2: 11.5

MySQL (InnoDB only): 8.

MariaDB : 10.5

PostgreSQL: 12, 15.

OS
*******

Should run under most Java-supporting OSes, but is actively supported under:

* Windows >= 2016
* Linux (tested under RHEL 8 or Ubuntu 22)
* AIX 7
